## Apa itu Web Programming?

Web programming adalah programming khusus untuk membuat konten yang bisa diakses lewat **browser** — website, aplikasi web, dan API.

---

## Cara Kerja Web

```
Kamu ketik URL         Browser kirim        Server proses
di browser        →    HTTP Request     →   & cari file HTML
                                                   ↓
Halaman tampil    ←    Browser render   ←   Server kirim Response
di layar               HTML/CSS/JS          (file HTML, CSS, JS)
```

---

## Frontend vs Backend

### Frontend (yang kita pelajari di kursus ini)
Bagian yang **dilihat dan disentuh** oleh user.

- **HTML** — struktur konten
- **CSS / Tailwind** — tampilan visual
- **JavaScript** — logika & interaksi

### Backend (level lanjutan)
Bagian yang berjalan di **server**, tidak terlihat user.

- **Database** — menyimpan data (PostgreSQL, MySQL)
- **Server Logic** — memproses request (Node.js, Python, Rust)
- **API** — jembatan data antara frontend dan backend

---

## Analogi Sederhana

> Website itu seperti **restoran**:
> - **Frontend** = ruang makan, menu, tampilan — yang dilihat pelanggan
> - **Backend** = dapur, gudang bahan — yang tidak terlihat pelanggan
> - **API** = pelayan yang membawa pesanan dari meja ke dapur

---

## Kenapa Belajar Web Dulu?

1. **Universal** — bisa diakses dari HP, laptop, tablet, TV
2. **Tidak perlu install** — cukup browser + internet
3. **Mudah di-deploy** — cukup upload ke hosting
4. **Banyak demand** — hampir semua bisnis butuh website
5. **Fondasi karir digital** — semua jalur (React, Node.js, dll) mulai dari sini
