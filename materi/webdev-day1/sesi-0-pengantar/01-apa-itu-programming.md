**PareLabs Web Dev · Batch 1 · Sesi 0 — Pengantar**

## Definisi

Programming (pemrograman) adalah proses menulis serangkaian instruksi yang dapat dimengerti dan dijalankan oleh komputer untuk menyelesaikan suatu tugas.

> **Analogi:** Bayangkan kamu menulis resep masakan yang sangat detail — "ambil 2 butir telur, pecahkan di mangkuk, kocok 30 detik." Komputer adalah chef yang mengikuti resep itu **persis** kata per kata, tanpa improvisasi.

---

## Komputer Itu Pintar tapi Tidak Berinisiatif

- Komputer dapat memproses miliaran instruksi per detik
- Tapi komputer **tidak bisa berpikir sendiri**
- Mereka hanya mengikuti instruksi yang kita tulis
- Kalau instruksinya salah → hasilnya salah (garbage in, garbage out)

## Kenapa Belajar Programming?

Setiap aplikasi digital yang kamu pakai dibuat oleh programmer:

| Aplikasi | Dibuat Dengan |
|----------|--------------|
| Instagram, TikTok | Python, React Native |
| WhatsApp | Erlang, React Native |
| Google Maps | C++, JavaScript |
| Website PareLabs | Rust, Next.js |

**Programming = kemampuan membuat hal-hal digital dari nol.**

## 3 Bahasa Web yang Akan Kita Pelajari

```html
<!-- HTML — Struktur (kerangka bangunan) -->
<h1>Halo, saya belajar coding!</h1>
```

```css
/* CSS — Tampilan (cat dan dekorasi) */
h1 {
  color: purple;
  font-size: 32px;
}
```

```javascript
// JavaScript — Perilaku (logika dan interaksi)
document.querySelector('h1').addEventListener('click', function() {
  alert('Kamu klik judulnya!');
});
```
