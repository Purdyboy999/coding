## Apa itu Vibe Coding?

Istilah yang dipopulerkan **Andrej Karpathy** (mantan Director of AI, Tesla & OpenAI) pada Februari 2025.

**Vibe Coding** = pendekatan coding di mana developer fokus pada *intent* (maksud/tujuan) bukan *syntax* (cara teknis menulis kode). Kamu "merasakan" apa yang ingin dibuat, lalu AI yang menterjemahkan ke kode nyata.

---

## Contoh Nyata

### Cara Lama
```
1. Buka dokumentasi HTML
2. Cari syntax yang tepat
3. Ketik tag satu per satu
4. Test di browser
5. Ulangi jika ada error
```

### Cara Vibe Coding
```
Prompt ke AI:
"Buat navbar dengan logo PareLabs di kiri, menu Home, 
 Program, Kontak di kanan, dan tombol hamburger 
 yang muncul di mobile"

→ AI langsung menghasilkan HTML + CSS + JS yang lengkap
→ Kamu test, tweak jika perlu, selesai
```

---

## Contoh Prompt yang Efektif

```
KURANG BAIK:
"Buat navbar"

LEBIH BAIK:
"Buat navbar HTML dengan:
- Logo teks 'PareLabs' di kiri dengan warna ungu
- Menu: Home, Program, Tentang, Kontak
- Tombol 'Daftar Gratis' di paling kanan warna hijau
- Responsive: menu collapse jadi hamburger icon di mobile
- Gunakan Tailwind CSS"
```

**Semakin detail deskripsinya → semakin baik hasilnya.**

---

## Kelebihan Vibe Coding

✅ Sangat cepat untuk prototyping  
✅ Pemula langsung bisa melihat hasil nyata  
✅ Fokus ke desain dan logika, bukan syntax  
✅ Mengurangi frustasi karena syntax error  
✅ Bisa buat hal kompleks tanpa harus master semua teknologi  

---

## Keterbatasan Vibe Coding

⚠️ Kode AI bisa salah atau tidak optimal  
⚠️ Tanpa dasar HTML/CSS/JS, sulit memahami kode yang dihasilkan  
⚠️ Susah debug kalau tidak paham fundamentalnya  
⚠️ Untuk project skala besar, tetap butuh skill manual  
⚠️ AI bisa "hallucinate" — membuat kode yang terlihat benar tapi tidak fungsi  

---

## Kesimpulan

> Vibe coding adalah **akselerator**, bukan pengganti skill.
> 
> Di kursus ini:
> - Hari 1–4: kita pelajari **fundamentalnya** (HTML, CSS, JS)
> - Di setiap sesi: kita juga **praktik pakai AI** untuk mempercepat
> - Tujuannya: kamu bisa vibe code **dengan pemahaman**, bukan asal generate
