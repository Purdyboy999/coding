## Cara Pakai

### Untuk Mentor
1. Buka folder di VS Code: `File → Open Folder → webdev-day1`
2. Install ekstensi **Markdown Preview Enhanced** untuk baca file `.md`
3. Buka file `.html` → klik kanan → **Open with Live Server**
4. Ajarkan topik dari `.md` → demo langsung di `.html`

### Untuk Peserta
1. Download dan ekstrak folder ini
2. Buka di VS Code atau Firebase Studio
3. Ikuti materi dari `.md` dulu (teori)
4. Lalu buka `.html` di browser (praktik)
5. Coba ubah kode `.html` dan lihat perubahannya!
