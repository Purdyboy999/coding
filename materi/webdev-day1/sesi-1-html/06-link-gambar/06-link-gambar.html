Link & Gambar

1. Link ke Website Eksternal

🌐 Kunjungi parelabs.id (buka tab baru)

📚 MDN Web Docs — Referensi HTML terlengkap

💡 Selalu tambahkan rel="noopener noreferrer" saat menggunakan target="_blank" untuk keamanan.

2. Link ke Halaman Lain di Website

← Kembali ke Topik 05 (Teks & Heading)

3. Anchor Link — Scroll ke Bagian di Halaman Ini

Navigasi cepat:

  • ↓ Langsung ke bagian Gambar
  • ↓ Langsung ke bagian Figure
  • ↓ Langsung ke Latihan
💡 Klik link di atas, browser akan scroll ke elemen dengan id yang sesuai.

5. Gambar dengan <img>

Gambar di atas menggunakan src dari URL eksternal dan alt yang deskriptif.

6. Kenapa alt Penting?

⚠️ Gambar di atas sengaja rusak. Teks alt-nya muncul sebagai pengganti. Tanpa alt, user hanya lihat ikon gambar broken tanpa keterangan apapun.

7. Gambar sebagai Link

Gambar di atas adalah link — coba klik!

8. Figure & Figcaption

💡 <figure> + <figcaption> adalah cara semantik untuk gambar dengan keterangan. Google menggunakannya untuk memahami konteks gambar.

Latihan

  1. Buat navigasi dengan 4 link: Home, Program, Tentang, Kontak
  2. Buat link ke parelabs.id yang terbuka di tab baru (dengan rel yang tepat)
  3. Tambahkan gambar dari internet (bisa pakai placeholder.com)
  4. Buat gambar yang juga berfungsi sebagai link
  5. Buat <figure> dengan gambar dan <figcaption>
  6. Tambahkan link email dan telepon untuk section kontak
