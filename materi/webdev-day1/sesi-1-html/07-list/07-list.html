List (Daftar)

1. Unordered List <ul> — Daftar Bullet

Materi yang akan dipelajari di kursus ini:

  • HTML — struktur halaman web
  • CSS — tampilan dan layout
  • Tailwind CSS — framework CSS utility
  • JavaScript — logika dan interaksi
  • SEO On-Page — optimasi mesin pencari
  • Hosting — deploy website ke internet
💡 Gunakan <ul> kalau urutan item tidak penting.

2. Ordered List <ol> — Daftar Nomor

Langkah membuat halaman HTML pertama:

  1. Buka VS Code atau Firebase Studio
  2. Buat file baru bernama index.html
  3. Ketik ! lalu tekan Tab untuk struktur dasar
  4. Tulis konten di dalam <body>
  5. Simpan file dengan Ctrl+S
  6. Buka di browser — lihat hasilnya!
💡 Gunakan <ol> kalau urutan item penting — langkah-langkah, ranking, tutorial.

3. Tipe Penomoran <ol>

type="1" (default)

  1. HTML
  2. CSS
  3. JavaScript

type="A"

  1. HTML
  2. CSS
  3. JavaScript

type="I" (romawi)

  1. HTML
  2. CSS
  3. JavaScript

4. Description List <dl> — Daftar Definisi

Glosarium istilah web development:

HTML
HyperText Markup Language — bahasa untuk membuat struktur halaman web
CSS
Cascading Style Sheets — bahasa untuk mengatur tampilan dan layout
JavaScript
Bahasa pemrograman yang berjalan di browser untuk menambah interaktivitas
API
Application Programming Interface — cara dua program berkomunikasi satu sama lain

6. Penggunaan Nyata — Navigation Menu dari <ul>

Navbar horizontal (ul + li + a + CSS flex):

  • Home
  • Program
  • Tentang
  • Blog
  • Daftar Gratis

Sidebar navigation (ul + li + a + CSS):

  • 📋 Dashboard
  • 🎓 Program Saya
  • 📚 Materi
