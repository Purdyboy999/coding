02 · Struktur Dasar HTML

Halaman ini mendemonstrasikan struktur dasar HTML yang benar. Buka kode sumber halaman ini (Ctrl+U) untuk melihat setiap bagiannya.

Apa yang ada di <head>?

Tag <head> di atas berisi:

  • charset UTF-8 — agar karakter Indonesia tampil dengan benar
  • viewport — agar tampil baik di HP dan tablet
  • description — muncul di deskripsi Google
  • title — teks di tab browser (lihat tab kamu!)
  • style — CSS internal untuk tampilan halaman ini
💡 Semua ini tidak terlihat di halaman, tapi sangat penting untuk SEO, tampilan di HP, dan pengalaman user.

Apa yang ada di <body>?

Semua yang kamu lihat di halaman ini ada di dalam tag <body> — heading ini, paragraf ini, section ini, dan footer di bawah.

💡 Rule: Kalau kamu ingin sesuatu terlihat di halaman → tulis di dalam <body>

Cek: Apakah HTML-mu Valid?

Kamu bisa validasi HTML di:

validator.w3.org — tools resmi W3C untuk cek validitas HTML

HTML yang valid → lebih baik di SEO, lebih konsisten di semua browser.

Latihan

  1. Ganti isi tag <title> → lihat perubahan di tab browser
  2. Ganti isi meta description → bayangkan ini muncul di Google
  3. Coba hapus charset UTF-8 → ketik karakter Indonesia → apa yang terjadi?
  4. Coba hapus viewport meta → buka di developer tools mobile view → apa yang terjadi?
