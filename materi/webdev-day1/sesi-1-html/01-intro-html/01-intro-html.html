Halo! Ini adalah halaman HTML pertamaku

File ini adalah contoh paling dasar dari HTML. Semua yang kamu lihat di sini adalah hasil dari tag-tag HTML yang ditulis di dalam file 01-intro-html.html ini.

Apa yang sedang terjadi?

Browser kamu sedang membaca file ini dari atas ke bawah, lalu merender setiap tag HTML menjadi tampilan visual yang kamu lihat sekarang.

Tiga pilar web

Website dibuat dari tiga teknologi:

<HTML> menentukan struktur (apa yang ada di halaman)
{CSS} menentukan tampilan (bagaimana terlihatnya)
JS() menentukan perilaku (apa yang terjadi saat interaksi)

Coba klik kanan → "View Page Source"

Kamu akan melihat kode HTML mentah yang sedang dirender oleh browser. Inilah yang sebenarnya "dikirim" oleh server ke browser kamu!

Tugas micro

1. Buka file ini di VS Code atau Firebase Studio
2. Ganti teks di dalam <h1> dengan nama kamu
3. Simpan file (Ctrl+S)
4. Refresh browser → lihat perubahan!
