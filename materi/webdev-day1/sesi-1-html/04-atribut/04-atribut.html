Atribut HTML

Atribut id — Identitas Unik

Ini paragraf dengan id="paragraf-pertama".

Klik link ini untuk scroll ke section kontak: Ke bagian kontak ↓

💡 id harus unik — satu halaman satu id. Dipakai untuk link anchor dan JavaScript.

Atribut class — Kelompok Elemen

Kedua tombol di atas punya class berbeda tapi bisa diberi style dari CSS yang sama.

💡 class bisa dipakai di banyak elemen. Satu elemen boleh punya beberapa class sekaligus.

Atribut href — Link Tujuan

🌐 Ke website eksternal (buka tab baru)

📄 Ke file HTML lain di folder ini

⬇️ Ke bagian di halaman ini (anchor)

📧 Buka email client (mailto)

📞 Buka dial pad HP (tel)

Atribut src & alt — Gambar

Gambar di atas menggunakan src (URL gambar) dan alt (deskripsi).

⚠️ Gambar di atas sengaja rusak — lihat bagaimana alt ditampilkan sebagai gantinya.

Atribut style — CSS Inline

Teks merah besar tebal

Teks hijau miring

Kotak ungu dengan teks putih
⚠️ Style inline boleh tapi hindari berlebihan. Lebih baik pakai class CSS untuk konsistensi.

Atribut Form: type, placeholder, required, disabled

📍 Kamu sampai di bagian #kontak!

Ini adalah section dengan id="kontak". Link anchor href="#kontak" di atas akan scroll ke sini.

Latihan

  1. Buat link ke parelabs.id yang terbuka di tab baru
  2. Tambahkan gambar dengan atribut src dan alt
  3. Buat 2 tombol — satu dengan class "btn-primary", satu "btn-secondary"
  4. Buat form dengan input email yang required dan satu input yang disabled
  5. Tambahkan anchor link yang scroll ke section di bawah halaman
